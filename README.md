## Run Locally

### 1. Clone repo

```shell
$ git clone "https://github.com/PranavGPR/FullyCart.git"
$ cd FullyCart
```

### 2. Install MongoDB

Download it from here: https://docs.mongodb.com/manual/administration/install-community/

### 3. Run Backend

```shell
$ npm install
$ npm start
```

### 4. Run Frontend

```shell
# Open new terminal
$ cd app
$ npm install
$ npm start
```
